new project-name --here --plugin --unattended --select value --install-deps value

    Creates a new Aurelia application project.

    project-name (optional) - The name to create the project with.
    If one isn't provided, the wizard will ask for one.

    --here - Creates a new project with the current working
    directory as the root of the project instead of creating a new project
    folder.

    --plugin - Creates a new Aurelia plugin project.

    --unattended - Creates project in unattended mode. It will
    fail if there is any conflicting file in target folder.

    --select - Preselect features (e.g. --select
    cli-bundler,alameda,karma). In unattended mode, these overwrite
    default-esnext selections. In interactive mode, some questions will be
    skipped because they are pre-answered.

    --install-deps - Auto install dependencies using yarn or npm
    (e.g --install-deps yarn)

help

    Displays the Aurelia CLI help.
